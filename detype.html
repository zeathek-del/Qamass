1. តើ Data type នៅក្នុង PHP មានអ្វីខ្លះ?

>> នៅក្នុង PHP មាន Data Type សំខាន់ៗជា ៣ ក្រុមធំៗ និង ៨ ក្រុមតូចៗ៖

  1. ក្រុមទី១៖ Scalar Data Types
    • Integer – លេខគត់
    • Float (Double) – លេខទសភាគ
    • String – អក្សរ ឬពាក្យ
    • Boolean – តម្លៃពិត (true) ឬ មិនពិត (false)
  2. ក្រុមទី២៖ Compound Data Types
    • Array – ផ្ទុកទិន្នន័យច្រើនក្នុងអថេរតែមួយ
    • Object – ប្រើប្រាស់ជាមួយ Class និង Programming បែប OOP
  3. ក្រុមទី៣៖ Special Data Types
    • NULL – តំណាងឱ្យអថេរដែលគ្មានតម្លៃ
    • Resource – ប្រើសម្រាប់ទិន្នន័យក្រៅ (ឧទាហរណ៍៖ database connection)
